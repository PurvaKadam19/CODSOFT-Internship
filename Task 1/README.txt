# Job Board Website

## Project Name
Job Board Website

## Description
A responsive Job Board website where employers can post job openings and job seekers can browse available jobs. The project is built using HTML, CSS, and JavaScript.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Responsive design
- Home page
- Job listings
- Job details section
- Search functionality
- User-friendly interface
