# Online Quiz Maker

## Project Name
Online Quiz Maker

## Description
A responsive Online Quiz Maker website that allows users to create quizzes and attempt multiple-choice questions. It displays the final score after quiz completion.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Responsive design
- Multiple-choice questions
- Instant score calculation
- Interactive user interface
- Easy navigation
